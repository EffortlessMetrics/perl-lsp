# Support Policy

Thank you for using perl-lsp! We want to ensure you have the best experience possible. This document outlines how to get help and what you can expect from the maintainers.

## How to Get Help

Before reaching out, please check the existing documentation:

- **[Frequently Asked Questions (FAQ)](docs/FAQ.md)**: Common questions and issues.
- **[Troubleshooting Guide](docs/TROUBLESHOOTING.md)**: Solutions for common setup problems.
- **[Editor Setup](docs/EDITOR_SETUP.md)**: Detailed configuration for various editors.
- **[Documentation Index](docs/README.md)**: Comprehensive guides and references.

### 1. GitHub Discussions

For general questions, ideas, or to show off how you're using perl-lsp, please use [GitHub Discussions](https://github.com/EffortlessMetrics/perl-lsp/discussions).

### 2. GitHub Issues

If you've found a bug or have a feature request, please search the [existing issues](https://github.com/EffortlessMetrics/perl-lsp/issues) first. If it hasn't been reported:

- **Bug Reports**: Use the [Bug Report template](https://github.com/EffortlessMetrics/perl-lsp/issues/new?template=bug_report.yml).
- **Feature Requests**: Use the [Feature Request template](https://github.com/EffortlessMetrics/perl-lsp/issues/new?template=feature_request.yml).
- **Parser Issues**: For specific Perl syntax parsing errors, use the [Parser Bug template](https://github.com/EffortlessMetrics/perl-lsp/issues/new?template=parser_bug.yml).

### 3. Security Vulnerabilities

**DO NOT** open public issues for security vulnerabilities. Please follow our [Security Policy](SECURITY.md) and report them privately.

## Service Level Agreement (SLA)

As a v0.9.x alpha release, we target the following response times for issues:

| Priority | Response Time | Fix Time (Target) |
|----------|---------------|-------------------|
| **P0 - Critical** (Crash, Data Loss) | 24 hours | 72 hours |
| **P1 - High** (Core feature broken) | 48 hours | 1 week |
| **P2 - Medium** (Minor bug, workaround exists) | 1 week | 1 month |
| **P3 - Low** (Cosmetic, enhancement) | 2 weeks | 3 months |

*Note: These are targets for the maintainers and may vary based on complexity and availability.*

## Supported Versions

We only provide active support for the current major version and the immediate previous minor version.

| Version | Status | Support Level |
|---------|--------|---------------|
| **0.9.x** | **Current** | Full Support (Bug fixes + Security) |
| **0.9.x** | Maintenance | Security Patches Only |
| **< 0.9** | End of Life | No Support |
